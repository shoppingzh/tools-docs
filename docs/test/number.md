# number

## format

`format(value: number, thousandth?: boolean, fraction?: number): string`

- `value`
  - 普通数值
  - undefined/null
  - NaN
  - Infinity/-Infinity
  - string/boolean/function
- `thousandth`
  - undefined/null
  - true/false
  - 其他
- `fraction`
  - undefined/null
  - 大于0
  - 小于0
  - 大于20位
