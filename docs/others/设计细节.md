## dom

### createWatermark

水印参数：

- 字体
- 文字颜色
- 文字大小
- 文字加粗
- 文字斜体
- 旋转角度
- 间距

绘制思路：

- 在canvas画布上绘制出水印内容（间距由留白控制）
- 导出图片的base64字符串
- 在页面创建一个占满全屏的元素，将导出的图片内容作为背景图，设置重复模式
- 在canvas上创建重复模式，将绘制了水印的canvas重复绘制在新canvas上
